## 五子棋网页小游戏

一个使用原生 HTML、CSS、JavaScript 实现的 15×15 五子棋小游戏，直接双击 `index.html` 即可离线游玩。界面采用暖色木纹风格，内置玩家 vs 玩家、玩家 vs AI 两种模式，并带有悔棋、状态提示和胜负高亮等辅助功能。

### 核心亮点
- **棋盘体验**：Canvas 绘制 15×15 网格与星位，自动保持正方形，提供交叉点悬停预览、最近一步高亮以及胜利连线特效。
- **控制面板**：右侧卡片式面板显示模式、当前轮次、总步数、AI 状态与结果提示；底部消息条用于展示模式切换或悔棋等动态信息。
- **悔棋机制**：人人对战每次撤销 1 步，人机对战每次撤销双方最近 2 步，落子记录以栈结构管理。
- **AI 逻辑**：简易启发式 AI——优先取胜、其次防守堵点，未命中时在已有棋子附近选择评分最高的位置，思考时展示加载动画。
- **自适应布局**：大屏保持“棋盘 + 控制面板”并排，小屏自动改为上下布局，按钮、下拉框与信息条均可换行以防压缩。

### 使用方式
1. 克隆或下载本仓库。
2. 双击打开 `index.html`（无需任何构建或服务器）。
3. 在控制面板中选择模式即可开始对弈。

### 操作指南
- **落子**：点击棋盘交叉点即可落子，黑棋先手；已有棋子处会提示不可落子。
- **模式切换**：通过下拉框在“玩家 vs AI”与“玩家 vs 玩家”之间切换，切换后自动开局。
- **重新开始**：清空棋盘、重置状态，重新由黑棋先手。
- **悔棋**：根据当前模式撤销最近棋步，按钮在无可撤销时会禁用。
- **提示信息**：状态卡片与消息条实时显示轮次、结果、AI 状态及系统提示。

### 开发说明
- 所有逻辑集中于 `index.html`，未引入任何第三方库。
- 主要结构划分为：数据状态、渲染、事件绑定、AI 逻辑四个部分，并辅以中文注释说明函数用途和参数。
- 若需扩展功能（如网络对战、更复杂 AI），可在现有状态管理与渲染框架基础上迭代。
